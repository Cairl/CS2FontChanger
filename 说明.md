## 🎮 概述

CS2 字体更改器让您能够轻松自定义 Counter-Strike 2 游戏内的字体显示。无论您是为了提高可读性还是追求个性化的外观，这款工具都能简单安全地满足您的需求。

## ⭐ 特色

- 🎯 **简洁易用** - 下载后将字体文件拖拽至软件即可开始运行
- 🔍 **智能检测** - 自动定位游戏安装路径，自动处理字体文件命名
- 🎨 **完整集成** - 自定义游戏内全部文字元素，且支持字体大小调节
- 🛡️ **安全可逆** - 首次运行自动备份，随时还原默认字体
- 🌐 **双语适配** - 依据系统语言自动切换中/英文界面

## 🚀 使用

1. 下载最新版程序
2. 准备好 `.ttf` 或 `.otf` 格式的字体文件
3. 将字体文件拖拽至程序图标，或启动后按界面提示导入
4. 按下[回车键]应用，享受全新游戏字体！
